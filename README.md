# Overview

This web application allows users to create a personalized weather dashboard by adding multiple cities to track their current weather conditions. The dashboard displays weather data including temperature, humidity, weather conditions, and icons for each selected city using the OpenWeather API.

The purpose of this website is to use javascript to connect to an API and then use the response data to display information in the website.

[Software Demo Video](https://www.loom.com/share/bbee94afc16241928f5e0716211a5352)

# Development Environment

- VSCode was used to createe this website
- Website was built using html, javascript and css
- OpenWeather API was used to get the weather data
- Google font Schibsted Grotesk was used for the current temperature
- Library Day.js was used to format the last updated date
